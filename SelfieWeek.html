Reflectie:
Het was erg leuk en interessant om aan mijn eigen project te werken. Ik vond het leuk om zelf te kunnen beslissen wat ik wou doen met mijn project. Ik was erg enthousiast over mijn project maar ik merkte wel dat ik mijn verwachtingen te hoog had gelegd. Ook had ik beter kunnen voorzien dat we minder dan een week hadden voor het project (i.v.m. een leraren dag). Ik heb geleerd hoe het is om met een groot media project op mezelf te zijn, dat ik mijn verwachtingen niet te hoog moet leggen en dat ik altijd moet klaar staan met een dag kwijt raken in een project of het nou door ziekte, een crashende computer of door een onverwachte deadline. Mijn idee was om een digitaal sprookje te maken. Je kan mijn hele gedachte gang en de feedback die ik van verschillende leraren heb gekregen volgen in mijn Selfie Week Diary maar omdat ik snap dat ik erg veel notities heb gemaakt moet het genoeg zijn om mijn reflecties te lezen.

Reflectie:
Ik ben begonnen met verschillende sprookjes te lezen voor inspiratie. Ik had al een groot voordeel met inspiratie zoeken omdat ik heel mijn leven al veel van sprookjes heb gehouden. Daarna heb ik het sprookje geschreven dit heb ik in het engels gedaan omdat engels een wereld taal is. Maar ik ben later nog verder gegaan met het itereren van covers.
