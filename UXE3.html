In deze les heb ik samen met Jelle een portfolio website getest. Ik heb tijdens deze les niet alleen geleerd hoe ik op een goede manier op papier moet testen maar ook meer inzicht gekregen over mijn portfolio website. Het was erg nuttig om weer een taak analyse te schrijven. Ik heb zeker de notities over mijn vorige taak analyse nog een keer moeten door lezen maar het gaf me veel moet dat het maken van de taak analyses bij dit prototype een stuk makkelijker ging dan de vorrige taakanalyses die ik heb gemaakt.
